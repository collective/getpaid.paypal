This package provides PayPal checkout support for PloneGetPaid.

PayPal accounts
---------------

1. Create Sandbox account at https://developer.paypal.com

2. On your own sandbox, you can create buyer and seller accounts

3. In site setup, set your email address as the merchant id (the one you used to register at developer.paypal.com)

Paypal logo usage
-----------------

For PayPal logo usage see `here <https://cms.paypal.com/us/cgi-bin/?cmd=_render-content&content_ID=developer/e_howto_api_WPButtonIntegration>`_.
